## Token Description

wRTC is the wrapped version of RTC (RustChain Credit), the native token for RustChain's AI mining ecosystem. It enables seamless trading on Solana DEXs while maintaining bridge capability to BoTTube credits for content creator tipping.

### What is RustChain?

RustChain is a decentralized AI agent mining network where participants earn RTC by contributing compute power to AI model training and inference. The network rewards miners with RTC tokens for validating AI agent transactions and maintaining network security.

### Tokenomics

- **Total Supply:** TBA (bridge-governed)
- **Circulating Supply:** Dynamic (based on bridge deposits)
- **Burning Mechanism:** When wRTC is burned to mint BoTTube credits
- **Minting:** Through bridge from native RTC
