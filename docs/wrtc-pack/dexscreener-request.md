## Token Description

wRTC is the wrapped version of RTC (RustChain Credit), the native token for RustChain's AI mining ecosystem. It enables seamless trading on Solana DEXs while maintaining bridge capability to BoTTube credits for content creator tipping.

RustChain is a decentralized AI agent mining network where participants earn RTC by contributing compute power to AI model training and inference. wRTC provides liquidity and accessibility to the broader Solana ecosystem.

## Use Cases

1. **AI Mining Rewards:** Earned by running RustChain miners
2. **Content Tipping:** Bridge wRTC to BoTTube credits for tipping creators
3. **Trading:** Trade on Raydium and other Solana DEXs
